# Kwizdb

Kwizdb is a Rust-based DBMS designed for experimentation.

## Commit Convention

**Format:** `<type>(<scope>): <subject>`

**Types:** `feat`, `fix`, `docs`, `style`, `refactor`, `perf`, `test`, `chore`, `build`, `ci`

**Examples:**

```
feat(index): add B+ tree index support
fix(query): resolve NULL handling in WHERE clause
test(storage): add unit tests for page eviction
chore: update Rust dependencies
```
